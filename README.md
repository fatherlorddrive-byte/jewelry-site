# Royal Crest Gems - Luxury Jewelry Website

> Where Elegance Meets Legacy | Est. 2011

A premium luxury jewelry e-commerce website built with Next.js, featuring handcrafted collections of fine gold, diamonds, watches, and precious gemstones with showrooms in the United States and Sweden.

## Features

- Responsive design with warm luxury aesthetic (gold, champagne, cream palette)
- Categorized product collections with subcategories
- CEO profile section
- Dual-location showroom information with Google Maps
- Contact form with mailto integration
- FAQ section
- SEO optimized with metadata on every page

## Tech Stack

- **Framework:** Next.js 15 (Static Export)
- **Styling:** Vanilla CSS with custom design tokens
- **Fonts:** Playfair Display + Lato (Google Fonts)
- **Deployment:** Vercel

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Deployment

This site is configured for static export and optimized for Vercel deployment.
